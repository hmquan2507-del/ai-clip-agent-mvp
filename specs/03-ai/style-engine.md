# Style Engine

Status: Active
Owner: Ho Quan
Last Updated: 2026-07-01
Related Epic: EPIC-02 Product Specification

---

## Purpose

Define how reusable production styles control AI editing behavior.

A style is not only a visual theme. A style is a production recipe.

---

## Style Controls

Each style defines:

- Subtitle
- Motion
- Zoom
- B-roll
- Sound FX
- Music
- CTA
- Pacing
- Hook tone
- Caption tone

---

## Initial Style Categories

- Talking Head
- Podcast
- Education
- Business
- Finance
- Storytelling
- Gaming
- Luxury
- Real Estate

---

## Style Definition Shape

Each style should eventually include:

```text
Name
Target user
Best content type
Subtitle behavior
Motion behavior
B-roll behavior
Music behavior
Sound FX behavior
CTA behavior
Render preset
Prompt instructions
```

---

## MVP Style Priority

1. Talking Head
2. Podcast
3. Education
4. Business

These match the earliest product users and current render direction.

---

## Non-goals

- Full marketplace of styles in MVP
- Complex visual editor for styles in MVP
- User-created animation systems in MVP
