# Purpose

Asset Library is the central repository of reusable media resources used by AI Clip Agent.

Instead of generating every visual or audio element through AI, the platform reuses curated assets.

This approach:

- reduces AI cost
- increases editing quality
- improves rendering speed
- ensures visual consistency
