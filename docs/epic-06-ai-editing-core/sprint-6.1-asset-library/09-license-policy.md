# License Policy

Status: Draft

Version: 1.0

---

# Purpose

Every asset inside AI Clip Agent must have a clearly defined license.

Assets without valid licensing information must not be available for production use.

# Import Rules

Every imported asset must pass:

License validation

Virus scan

Metadata extraction

Quality review

---

# Expired Assets

Expired assets must automatically become unavailable for AI matching.

---

# User Assets

User uploaded assets remain owned by the user and are isolated from the global Asset Library unless explicit sharing is enabled.
