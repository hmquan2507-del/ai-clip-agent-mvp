# Asset Taxonomy

Status: Draft

Version: 1.0

---

# Purpose

Asset Taxonomy defines the hierarchical classification system for every reusable asset inside AI Clip Agent.

The taxonomy enables consistent organization, fast search, scalable growth, and accurate AI matching.

Every asset must belong to exactly one Asset Type and one primary Category, while supporting multiple tags and keywords.
