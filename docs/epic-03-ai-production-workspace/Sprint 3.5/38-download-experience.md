## Purpose

Define how users access completed exports.

## Download UI

The completed export screen should show:

- video preview thumbnail
- file name
- duration
- resolution
- file size
- created time
- download button

## User Actions

Users can:

- download video
- copy link if supported
- view export history
- create another export
- return to workspace

## Rules

Download should only be available after render completion.

If the file is expired or missing, show recovery guidance.
