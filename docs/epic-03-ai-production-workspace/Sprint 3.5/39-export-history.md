## Purpose

Define export history for a Production.

Export history helps users track previous rendered outputs.

## History Item Fields

Each export history item includes:

- export id
- version
- status
- format
- resolution
- created time
- file size
- download availability

## User Actions

Users can:

- download previous export
- view export details
- delete export if allowed
- create new export version

## Future Considerations

Export history may later support:

- platform presets
- public share links
- expiration policy
- team access control
