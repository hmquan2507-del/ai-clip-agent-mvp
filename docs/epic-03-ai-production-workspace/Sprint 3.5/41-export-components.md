## Purpose

Define reusable UI components for Export Experience.

## Components

- Export Settings Panel
- Export Summary Card
- Render Progress Card
- Download Card
- Export History List
- Export Error Banner
- Retry Export Button
- Cancel Render Button

## Component Rules

Components must support:

- loading state
- disabled state
- error state
- completed state

## Primary CTA

The primary CTA before rendering is:

```text
Start Export
```

After completion:

```text
Download Video
```
