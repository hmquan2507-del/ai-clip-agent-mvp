## Purpose

Sprint 3.5 defines the Export Experience for AI Clip Agent.

Export is the final step after the user reviews and approves AI-generated output.

## Primary Flow

```text
Approved Production
    ↓
Choose Export Settings
    ↓
Start Render
    ↓
Track Render Progress
    ↓
Download / Share Export
```
