## Purpose

Define reusable UI components used by Review Workspace.

## Components

Core components:

- Review Header
- Video Preview Player
- Transcript Panel
- Subtitle Preview
- AI Suggestion Card
- Quality Badge
- Comment Thread
- Action Toolbar
- Regenerate Modal
- Approval Modal
- Empty State
- Error State

## Component Ownership

Components belong to Review Workspace but should follow the global design system.

## Component Rules

Components must:

- be reusable
- support loading state
- support error state
- support empty state
- be keyboard accessible

## Primary Action Toolbar

Recommended actions:

- Approve
- Regenerate
- Comment
- Export
- Back to Production Center
