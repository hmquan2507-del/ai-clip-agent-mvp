## Purpose

Define keyboard shortcuts for faster review.

Shortcuts should speed up review without making the product feel like a complex editor.

## Core Shortcuts

| Shortcut | Action |
|---|---|
| Space | Play / Pause |
| Left Arrow | Seek backward |
| Right Arrow | Seek forward |
| J | Previous clip |
| K | Play / Pause |
| L | Next clip |
| A | Approve |
| R | Regenerate |
| C | Add comment |
| / | Search transcript |

## Rules

- Shortcuts must not conflict with text input.
- Shortcuts must be discoverable.
- Shortcuts must have visible help.
- Critical actions require confirmation.

## Accessibility

Keyboard navigation must support:

- video controls
- transcript panel
- suggestions
- primary action buttons
