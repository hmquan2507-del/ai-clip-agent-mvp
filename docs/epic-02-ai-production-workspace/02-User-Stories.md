# User Stories - AI Production Workspace

Status: Active
Owner: Ho Quan
Last Updated: 2026-07-01
Related Epic: EPIC 2.2

---

## Home

As a user, I want to see my recent productions so I can continue work quickly.

As a user, I want a clear Start Production button so I can begin without thinking.

As a user, I want Home to show production status, not admin charts.

---

## Upload

As a user, I want to upload a source video so AI can process it.

As a user, I want to choose a style so AI knows the editing direction.

As a user, I want to add a simple objective so AI can optimize hooks and clips.

As a user, I want validation errors to be clear if my file cannot be used.

---

## AI Queue

As a user, I want to see the AI processing stages so I know work is happening.

As a user, I want to know which stage failed if something breaks.

As a user, I want to retry failed stages instead of starting over.

---

## Review

As a user, I want to preview generated clips so I can choose the best ones.

As a user, I want to see why AI selected a clip.

As a user, I want to approve, reject, or regenerate clips.

As a user, I want to make small text corrections without opening a complex editor.

---

## Export

As a user, I want to export approved clips as social-ready videos.

As a user, I want to see render progress.

As a user, I want to download completed outputs.

---

## Admin / Settings

As a user, I want settings to be separate from the production flow.

As a user, I want API/storage/provider setup to not distract from normal production work.
