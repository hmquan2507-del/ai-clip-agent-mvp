# EPIC 2.2 - AI Production Workspace

Status: In Progress
Owner: Ho Quan
Last Updated: 2026-07-01

---

## Goal

Build the first product workspace for AI Clip Agent.

This workspace must follow the production flow:

```text
Home
  ->
Upload
  ->
AI Queue
  ->
Review
  ->
Export
```

This is not an admin dashboard and not a timeline-first video editor.

---

## Scope

Included:

- Product PRD
- User stories
- UX flow
- UI direction
- Component plan
- API contract
- Data model
- Implementation tasks
- Acceptance criteria
- Style Engine notes
- AI Pipeline notes

---

## Workspace Principle

The user should not edit from scratch.

The user should upload a video, wait for AI output, review clips, approve or regenerate, then export.
