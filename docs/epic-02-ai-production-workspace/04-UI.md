# UI - AI Production Workspace

Status: Active
Owner: Ho Quan
Last Updated: 2026-07-01
Related Epic: EPIC 2.2

---

## UI Direction

Build a dark, focused AI production workspace.

Each route should show one clear job to be done.

---

## Navigation

Primary navigation:

- Home
- Upload
- AI Queue
- Review
- Export
- Productions
- Styles
- Settings

Each item should navigate to a page, not scroll to a section.

---

## Home UI

Show:

- Start Production button
- Active production cards
- Recent productions
- Quick status summary

Avoid:

- Admin-style metric overload
- Upload form inside Home
- Review queue inside Home

---

## Upload UI

Show:

- Upload dropzone
- Style selector
- Objective input
- File validation
- Start AI Production button

---

## AI Queue UI

Show:

- Production title
- Pipeline stages
- Current progress
- Error/retry state

---

## Review UI

Layout:

```text
Clip list | Preview | AI plan/actions
```

Show:

- Clip preview
- Hook
- Caption
- CTA
- AI reason
- Approve / Reject / Regenerate

---

## Export UI

Show:

- Approved clips
- Render status
- Download actions
- Output format

---

## Mobile Direction

Mobile should stack panels:

1. Header/status
2. Primary content
3. Actions
4. Secondary details
